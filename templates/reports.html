List of Available Reports & Descriptions


1 - DAGR Metadata Query

Typically, we would like to write queries to find DAGRs using the metadata attributes, such as date, author, type, size, or keywords in the annotations.

2 - Orphan and Sterile Reports

Generate all the DAGRs that are not referenced by a parent DAGR or have no descendant DAGRs.

3 - Reach Queries

Very often we would like to find all the DAGRs that can be reached by a given DAGR (descendants) or those that point to it (ancestors).

4 - Time-Range DAGR Report

Generate a report with several preselected data items for all the DAGRs created or entered in a given time range.
